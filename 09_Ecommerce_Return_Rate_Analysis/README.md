# E-Commerce Return Rate Analysis

Objective:
Analyze return behavior in an e-commerce business to identify key drivers of product returns, quantify financial impact, and recommend operational improvements.

Tools Used:
- Power BI (Dashboard & Data Modeling)
- MySQL (Exploratory Data Analysis)
- Python (Data Cleaning & Preparation)

Project Scope:
- Calculated return rate and revenue impact
- Analyzed return behavior by product category
- Evaluated shipping and payment method return patterns
- Identified top return reasons driving revenue loss

Key Metrics:
- Total Orders: 10,000
- Total Returns: 5,052
- Return Rate: 50.52%
- Lost Revenue: $1.27M
- Net Revenue: $1.25M

Key Insights:
- Over 50% of orders resulted in returns, significantly impacting profitability.
- Defective products and incorrect shipments were leading return drivers.
- Clothing and Books showed the highest return frequency.
- Next-Day shipping and Gift Card payments exhibited higher return rates.

Business Implications:
Findings suggest the need for quality control improvements, shipping accuracy monitoring, and targeted policy review for high-risk payment and delivery methods.
