# Adidas US Sales Performance Dashboard

Objective:
Develop an interactive Tableau dashboard to analyze Adidas US sales performance across regions, categories, and time periods (2020–2021).

Tools Used:
- Tableau
- Microsoft Excel (Data Preparation)

Dashboard Scope:
- Time-based sales trend analysis
- Regional revenue comparison
- Category-level performance evaluation
- Sales distribution analysis across segments

Key Insights:
- Q3 2021 recorded peak sales performance across regions.
- Western region generated the highest total revenue.
- Men’s Street Footwear emerged as the top-performing product category.
- Revenue contribution remained relatively balanced across major segments.

Features:
- Interactive filtering by region and category
- Dynamic trend visualization
- Comparative performance analysis
