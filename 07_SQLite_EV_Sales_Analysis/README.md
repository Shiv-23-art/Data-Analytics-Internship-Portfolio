# Electric Vehicle (EV) Sales Analysis

Objective:
Analyze electric vehicle sales performance to evaluate revenue trends, brand performance, discount impact, and overall sales growth using SQL and Python-based analytics.

Tools Used:
- SQLite
- Python (Pandas)
- Matplotlib
- Seaborn

Project Scope:
- Queried structured EV sales data using SQL
- Aggregated revenue and unit sales by brand and model
- Evaluated discount impact on revenue performance
- Conducted time-series analysis to identify sales growth patterns
- Visualized trends using bar charts, line plots, and heatmaps

Key Insights:
- Identified top-performing EV brands contributing highest revenue
- Observed correlation between discount strategies and sales volume
- Detected growth trends across time periods
- Highlighted models driving overall market performance

Technical Focus:
Emphasis on SQL-based data extraction, Python-driven transformation, and trend visualization to support business performance evaluation.
